File Management

Kamilah Louw

- OVERVIEW -

File and data management includes essential tasks like uploading, deleting, creating, and downloading files and folders. An improved interface was developed to simplify these tasks, speed up processes, and make the management of the files and data within the archives seamless. It is important that the interface be intuitive and straightforward for any user to limit the need for any prior knowledge.

- AIM -

The aim was to develop a file and data management system that was more intuitive, visually appealing, and included the basic functionality of standard file and data management systems.

- MAIN FEATURES -

- CONCLUSIONS -

The improved interface developed during this project not only includes the core functionality but also additional features and improvements such as batch-uploading, downloading, deleting, renaming files and folders, accepting zipped files, and creating folders directly through the system. The new interface proved to be more intuitive while speeding up file and data management processes.
